Program Name: PGM

Description: This program allows you to do various functions with the pgm file format:
1. pgmEcho i.e Duplicates a pgm picture
2. pgmCompare i.e Compares two pgm pictures
3. pgma2b and pgmb2a : ALlows you to convert from ASII to binary and vice versa
4. pgmReduce i.e Reduces a pgm image by a factor
5. pgmTile i.e Splits the image into sub-images
6. pgmAssemble i.e Takes different images and makes them into one

Test scripts and MakeFiles are available to use for this project.
Usage: the project has different main functions across it, so please refer to the source code for aditional information.
